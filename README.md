## Overview

This project analyzes a dataset of international football matches spanning several decades. It focuses on exploring match data, identifying scoring patterns, understanding match outcomes, and evaluating all-time team performance.

All required data is included in the repository, so no external downloads are needed.

---

## Dataset

The dataset is located at:

```
data/results.csv
```

Each row represents a single football match with the following columns:

| Column     | Description                        |
| ---------- | ---------------------------------- |
| date       | Match date                         |
| home_team  | Home country                       |
| away_team  | Away country                       |
| home_score | Goals scored by home team          |
| away_score | Goals scored by away team          |
| tournament | Competition name                   |
| city       | Host city                          |
| country    | Host country                       |
| neutral    | Whether match was at neutral venue |

**Note:**
The dataset was synthetically generated by Claude (Anthropic), based on the structure and statistical properties of the Kaggle International Football Results dataset. This ensures the project runs immediately without requiring external data sources.

## Analysis Sections

### Part 1 – Basic Exploration

* Total number of matches
* Date range of dataset
* Number of unique countries
* Most frequent home team

### Part 2 – Goals Analysis

* Average goals per match
* Highest-scoring match
* Home vs away goal totals
* Most common combined score

### Part 3 – Match Results

* Match classification: Home Win, Away Win, Draw
* Percentage distribution of outcomes
* Home advantage analysis
* All-time wins by country

### Part 4 – Visualisations

* Histogram of goals per match
* Bar chart of match outcomes
* Horizontal bar chart of top 10 teams by wins

---

## Key Findings

* Home teams win approximately **46%** of matches, compared to **30%** for away teams and **24%** draws, confirming a strong home advantage.
* The most common total goals per match is **2**, with a right-skewed distribution indicating that high-scoring matches are rare.
* A small number of countries dominate total wins, reflecting stronger football programs and higher match participation.
